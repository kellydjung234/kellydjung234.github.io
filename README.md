# LUWeather-a wherther website

### Function

---

LUheather strives to provide real-time city weather reports with the dark and light themes, which include :

- time and date
- temperatures
- air quality
    - PM2.5
    - PM10
    - S02
    - CO
    - NO
    - NO2
    - NH3
    - O3
- Humidity
- Cloudiness
- Pressure
- Visibility
- Wind speed
- Feel like
- 3-hourly temperatures with weather
- 7-day temperatures with weather
